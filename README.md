Welcome to the official backend service for the E-Commerce platform developed by **Dev-Teammm**. This system is built using **Spring Boot**, **PostgreSQL**, **Spring Security**, and **JPA**, providing a full-featured backend for an online shopping platform(e-commerce).

---

##  Features

### 1. User Account & Authentication
- Email, phone, and social login (Google/Facebook)
- Guest checkout support
- User profile & address management
- Password reset & recovery

### 2. üõí Product Management
- Product catalog with categories & subcategories
- Variants (size, color, etc.) with inventory tracking
- Media-rich descriptions (images/videos)
- Advanced search, filtering, and review system
- SKU and barcode support

### 3.  Cart & Wishlist
- Persistent shopping cart
- Wishlist/favorites feature
- Dynamic price calculation (with tax & discounts)

### 4.  Checkout & Payments
- Multi-step checkout process
- Saved addresses & order summaries
- Multi-currency, invoice generation
- Integration with payment systems ( Stripe,MOMO)

### 5. Shipping & Delivery
- Standard/express shipping options
- Order tracking and delivery scheduling
- Local pickup support

### 6.  Order Management
- Order history and status tracking
- Returns and cancellation system
- Admin controls: refunds, updates
- Email/SMS notifications

### 7. Search & Navigation functionalities
- Advanced search with autocomplete
- Category filters and sorting (price, popularity)
- Breadcrumb and navigation support
